## Setup / how to run

```bash
py -3.13 -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

Each folder is a self-contained set of scripts; run any script directly, e.g.:

```bash
python python-basics/01_data_structures.py
```
